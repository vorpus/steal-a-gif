# Sample clips

These power the "i gotchu fam" sample tiles in the landing chat. Each `.MP4` is
a short screen recording; the matching `.thumb.png` is its first frame (shown on
the tile).

- `tiktok-catfight.MP4` + `.thumb.png`
- `xhs-smelly.MP4` + `.thumb.png`
- `yt-snoop.MP4` + `.thumb.png`

Also used in the intro before/after demo:

- `xhs-smelly.before.png` — a square crop of the raw recording ("before")
- `example-export.gif` — the cleaned, transparent result ("after")

To regenerate a thumbnail from a new clip, grab its first frame and save it as
`<name>.thumb.png`.
